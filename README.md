# HangInThere: Rescue Glowy!

A fun, modern Hangman-style game featuring a cute monster named Glowy.

## Features
- **Modern UI**: Neon gradients, glassmorphism effects, and smooth transitions.
- **Cute Monster Illustration**: "Glowy" builds up piece-by-piece as you guess incorrectly.
- **Hint System**: Get a hint after 3 mistakes.
- **Countdown Timer**: 40 seconds to rescue Glowy!
- **Victory Facts**: Learn fun facts when you successfully guess the word.
- **Sound Effects**: Interactive audio feedback for guesses, wins, and losses.
- **Keyboard Support**: Works with both on-screen and physical keyboards.

## How to Play
1. Open `index.html` in any modern browser.
2. Guess the letters to reveal the hidden word.
3. Rescue Glowy before the 40-second timer runs out or you make 6 mistakes!

## Technologies used:
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- Web Audio API
- Canvas Confetti
